Данный проект доски объявлений для игрового форума выполнен согласно заданию школы SkillFactory в качестве итогового задания по бэкенду.

В проекте реализована возможность регистрации через емайл (приходит пятизначный код регистрации).
После регистрации пользователь может создавать объявления и отправлять отклики к объявлениям других игроков.

Также через отдельный url '/mailing' реализована возможность для администратора делать новостные рассылки.
олее подробно то, что реализовано, описано в отдельном файле с самим заданием 'task.txt'
